### step.2 宿泊代

【やること】  
- ホテル代
  - ホテルの基本料金は5000円
- オプション(料金上乗せ)
  - **必ず朝食と夕食、両方のオプションをホテル代に上乗せ**する
  - 朝食のみ
    - 500円
  - 夕食のみ
    - 800円
  - 朝食と夕食
    - 朝食代と夕食代を合わせた金額
- 合計金額を求める
  - 電車賃とホテル代の合計金額
  - 消費税は考慮しない
  

【期待する結果】  
php main.phpを実行すると
```
交通費： 20000円
ホテル代： 6300円
旅費合計： 26300円
```
と出力される
